# 功能开关与反馈

{% hint style="info" %}
`<>`表示参数任取；`[]`表示参数可以省略。
{% endhint %}

{% hint style="info" %}
除了通过bot的`来杯咖啡`反馈问题外，你也可以点击”[联系我](tencent://Message/?Uin=2423116072\&websiteName=www.oicqzone.com\&Menu=yes)“。
{% endhint %}

| 指令示例      | 功能说明          |
| --------- | ------------- |
| lssv      | 查看本群服务        |
| 启用 _服务名_  | 启用某一个lssv中的服务 |
| 禁用 _服务名_  | 禁用某一个lssv中的服务 |
| 来杯咖啡 _内容_ | 反馈bot问题       |
