## 功能命令和描述

### 维护组的命令

| 子模块    | 命令            |
| ------ | ------------- |
| 马娘数据库  | 手动更新马娘数据      |
| 马娘相性   | 手动更新相性信息      |
| 马娘抽卡   | 更新马娘卡池        |
| 马娘表情包  | 手动更新马娘表情包     |
| 马娘漫画   | 手动更新马娘漫画      |
| 马娘限时任务 | 手动更新限时任务      |
| 马娘技能   | 手动更新马娘技能      |
| 马娘新闻   | 马娘新闻翻译转发模式on  |
| 马娘新闻   | 马娘新闻翻译转发模式off |
